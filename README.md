# Lumina - Smart Study Planner

**Lumina** is a comprehensive, powerful vanilla JavaScript web application designed to streamline academic life for students everywhere. Built with a strong focus on clean architecture and local data persistence, it features a unique "Human-Made" design aesthetic that mimics the feel of a physical study journal. This project empowers users to organize courses, manage schedules, and track progress effortlessly without needing complex accounts or dependencies.

### Key Features
-   **Dashboard**: Detailed overview of subjects, tasks, and daily schedule.
-   **Subject Management**: Easily create, edit, and color-code academic courses.
-   **Schedule Planner**: Interactively plan study sessions with conflict detection to avoid overlaps.
-   **Task Manager**: Track assignments and exams with deadlines, priorities, and filtering.
-   **Progress Analytics**: Visualize study time distribution and task completion rates.
-   **Privacy Focused**: All data is stored locally in your browser using LocalStorage.

### Quick Start
1.  Clone this repository.
2.  Open `index.html` in any browser.
3.  Start organizing immediately.
