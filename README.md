# Lumina - Smart Study Planner

**Lumina** is a comprehensive, vanilla JavaScript web application designed to streamline academic life. Built with a focus on clean architecture and local data persistence, it features a unique "Human-Made" design aesthetic that mimics a physical study journal.

### Key Features
-   **Dashboard**: Get a high-level overview of subjects, pending tasks, and your daily schedule.
-   **Subject Management**: Organize courses with customizable color codes.
-   **Schedule Planner**: Interactively plan study sessions with visual conflict detection.
-   **Task Manager**: Track assignments and exams with deadlines, priorities, and status filtering.
-   **Progress Analytics**: Visualize study time and task completion rates.
